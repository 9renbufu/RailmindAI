## 二、余票查询数据流

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│  Vue3    │───>│ Gateway  │───>│  Ticket  │───>│  Redis   │
│  前端    │    │  限流    │    │  服务    │    │  缓存    │
└──────────┘    └──────────┘    └──────────┘    └──────────┘
                                    │   ▲              │
                                    │   │ 缓存未命中    │
                                    ▼   │              │
                               ┌──────────┐           │
                               │  MySQL   │<──────────┘
                               │  回源查询 │  写入缓存
                               └──────────┘
```

**缓存策略：**
- Key: `ticket:{trainNo}:{date}:{from}:{to}`
- Value: 座位类型 → 剩余数量 JSON
- TTL: 30秒（高峰期可缩短至10秒）
- 更新：库存变更时主动删除缓存（Cache Aside Pattern）
